# Aclas 电子称 Electron 插件

> 注：请保证机器有 Python 环境

## 环境要求
- node.js >= 12
- Python >= 3.0
- electron = 9.1.0

## 命令
- 启动 electron `npm run dev` or `yarn dev`
- 插件在 node 环境下测试 `npm run test` or `yarn test`
